# ArXiv Search MCP 使用示例

## 基础搜索

### 1. 按关键词搜索论文

**查询：** "搜索关于 transformer 的论文"

这将调用 `search_arxiv_papers` 工具，返回相关论文列表，包括：
- 论文标题
- 作者列表
- 发布日期
- 摘要预览
- PDF 和 arXiv 链接

### 2. 限制搜索结果数量

**查询：** "搜索关于 quantum computing 的前 5 篇论文"

可以通过 `max_results` 参数控制返回的论文数量。

### 3. 按时间排序

**查询：** "搜索最新的关于 GPT 的论文，按更新时间排序"

使用 `sort_by: "lastUpdatedDate"` 参数获取最新论文。

## 获取论文详情

### 通过 arXiv ID 获取完整信息

**查询：** "获取论文 2301.07041 的详细信息"

这将返回论文的完整信息：
- 完整摘要
- 所有作者
- 分类信息
- DOI（如果有）
- 期刊引用（如果有）
- 评论信息
